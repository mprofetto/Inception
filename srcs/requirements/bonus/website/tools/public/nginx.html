What is Nginx?

NGINX Open Source170 ou NGINX est un logiciel libre de serveur Web (ou HTTP) ainsi qu'un proxy inverse écrit par Igor Sysoev, dont le développement a débuté en 2002 pour les besoins d'un site russe à très fort trafic (Rambler). La documentation est disponible dans plusieurs langues. C'est depuis avril 2019, le serveur web le plus utilisé au monde d'après Netcraft171, ou le deuxième serveur le plus utilisé d'après W3techs172.
