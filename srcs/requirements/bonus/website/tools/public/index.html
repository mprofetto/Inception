What is Inception About

This project aims to broaden your knowledge of system administration by using Docker. You will virtualize several Docker images, creating them in your new personal virtual machine.

What are we building here ?

This build consist in building a full webserver infrastructure containing a webserver (NGINX), a wordpress website and his own php CGI manager and a database used by the wordpress website.

But what is really a webserver?

A webserver is a program running permanently and listening to request on one or several ports. Once the webserver receive a request on the listened port, the webserver decide if the request can be answered by sending an html document stored or if the request needs to be processed by a particular service.

Ok but how the webserver communicate with thoses particulars services?

The webserver use whats called a FastCGI (the modern version of a CGI). CGI stands for Common Gateway Interface. When the request has to be send through a pipe or a socket depending on the fact that the service wich will interpret the request is on the same machine as the webserv or on a different one.

And thats it?

Yeah thats it. Once the request has been processed and a response is given, the webserver send back the response to the client that send the request at the beginning (like an user navigator somewhere on the internet).
